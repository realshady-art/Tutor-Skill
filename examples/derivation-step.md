# Example: Derivation step

## User

Where does this derivation step come from?

## Assistant

Paste the two lines around that step. I need the local context first.

## User

From `a(b + c)` to `ab + ac`.

## Assistant

That step uses the **distributive law**.

The rule is:

`x(y + z) = xy + xz`

So here:

- `x = a`
- `y = b`
- `z = c`

which gives:

`a(b + c) = ab + ac`

So the change is not a new theorem. It is just applying one standard algebra rule to this exact line.
