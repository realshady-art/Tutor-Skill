# Eval Cases

These are the four baseline cases used during V1 review.

## 1. `What does acyclic mean?`

Expected behavior:

- explain the term directly in 1-2 sentences first
- add intuition in plain language
- give one tiny example
- do not drift into a long prerequisite lecture

## 2. `What is this slide talking about?`

Expected behavior:

- if no slide, screenshot, or excerpt is provided, ask for the material first
- if material is provided, explain the page goal before line-by-line detail
- do not guess missing content

## 3. `Where does this derivation step come from?`

Expected behavior:

- identify the exact rule, identity, or previous line being used
- explain the local step first
- do not automatically replay the whole derivation unless asked

## 4. `Walk me through one example.`

Expected behavior:

- set up one small example
- teach step by step
- explain why each step is legal
- end with the general rule or takeaway
